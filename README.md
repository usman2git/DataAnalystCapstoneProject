# 📊 Data Analyst Capstone Project  
### Stack Overflow Developer Survey 2024 — Full Analysis & Dashboard

This repository contains the complete end‑to‑end data analysis project based on the **2024 Stack Overflow Developer Survey**.  
It includes Python scripts (Parts 1–24), data cleaning, visualization, dashboard creation in Google Looker Studio, and the final presentation.


## 🧠 Project Overview

This project analyzes developer trends using the Stack Overflow 2024 dataset.  
It covers:

- Data cleaning & wrangling  
- Exploratory data analysis  
- Visualizations (bar charts, bubble charts, stacked charts, line charts, pie charts, maps)  
- Dashboard creation in **Google Looker Studio**  
- Final capstone presentation summarizing insights  

---

## 🛠 Tools & Technologies

- **Python**  
  - Pandas  
  - Matplotlib  
  - Seaborn  
- **Google Looker Studio**  
- **Jupyter Notebook**  
- **GitHub**  
- **CSV dataset (Stack Overflow 2024 subset)**
